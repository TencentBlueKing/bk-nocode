##### 简要描述

- 节点运行日志

##### 请求URL
- ` /openapi/log/ `
  
##### 请求方式
- GET

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ticket |是  |string| 单据  |
|state |是  |string| 节点id  |



##### 返回示例 

``` 
{
   {
    "result": true,
    "code": "OK",
    "message": "success",
    "data": {},
    ]
}
}
```
